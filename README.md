# Logue - A pythonic way of key-logging 
A keylogger that will log both keystrokes and mouse clicks. And it can only be stopped by either pressing Esc or a specific file in the file system. It is still in the developing phase so features might break easily. 

## Updates
New PII handler is made which scrapes the information from the system for further investigation and persistence.
